# Simple Console Expense Manager

## Overview of the Project

The **Simple Console Expense Manager** is a lightweight, command-line utility designed to track personal income and expenses. Built entirely in Python, it provides a clean, menu-driven interface for quickly logging transactions and viewing a real-time financial summary. This tool is perfect for users who need a no-fuss way to monitor their basic cash flow directly from the terminal.

---

##  Features

* **Income & Expense Tracking:** Easily add new transactions, classifying them as either 'Income' or 'Expense'.
* **Real-time Balance Calculation:** Automatically calculates the current net balance (Income - Expense).
* **Transaction History:** Displays a chronological list of all recorded transactions.
* **Visual Summary:** Presents a clear summary of total income, total expense, and net balance.
* **Input Validation:** Ensures only valid, positive numerical amounts are accepted for transactions.
* **Color-Coded Output:** Uses ANSI escape codes to display Income in green (✅) and Expenses in red (❌) for easy readability.
